# Level 0 to Level 1

## Goal
Connect to the Bandit server and read a simple file located in the home directory.

## Concept Explanation
The command line interface is text-based. You need to know how to connect to remote servers securely using SSH (Secure Shell) and how to output the contents of a file to your terminal screen.

## Commands Used
- `ssh`, `cat`, `ls`

## What I Learned
I learned the basics of establishing an SSH connection and viewing simple text files. It's the essential first step in remote server administration.
