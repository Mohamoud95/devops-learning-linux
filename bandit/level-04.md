# Level 3 to Level 4

## Goal
Find and read a hidden file inside a specific directory.

## Concept Explanation
In Linux, files and directories that start with a dot (`.`) are 'hidden' by default. You need specific flags to view them when listing directory contents.

## Commands Used
- `cd`, `ls -a`, `cat`

## What I Learned
I discovered how Linux handles hidden files and how to explore directories more thoroughly.
