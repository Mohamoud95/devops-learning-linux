# Level 11 to Level 12

## Goal
Decode a string that has been obfuscated using a ROT13 cipher.

## Concept Explanation
ROT13 is a simple letter substitution cipher. You can translate characters from one set to another using text translation command-line tools.

## Commands Used
- `tr`

## What I Learned
I learned how to perform basic character translation and manipulation on text streams.
