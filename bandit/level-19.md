# Level 18 to Level 19

## Goal
Log into a server where the standard shell immediately kicks you out, preventing normal interaction.

## Concept Explanation
SSH allows you to execute commands directly without requesting an interactive shell session. This is useful for bypassing restricted environments or automating remote commands.

## Commands Used
- `ssh` with remote command execution

## What I Learned
I discovered that SSH is not just for interactive terminal sessions, but also a powerful tool for remote command execution.
