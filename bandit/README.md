# OverTheWire: Bandit Journey

Welcome to my OverTheWire Bandit learning journey! This repository section documents my practical hands-on experience learning Linux command-line fundamentals and cybersecurity concepts.

## What is Bandit?

[Bandit](https://overthewire.org/wargames/bandit/) is a beginner-friendly wargame provided by the OverTheWire community. It is designed to teach the basics of Linux command-line navigation, file permissions, and basic scripting. Each "level" presents a challenge where you must use command-line tools to discover a hidden password to access the next level via SSH.

## Why I Used It

I chose to complete the Bandit challenges because they offer a highly practical, interactive way to solidify theoretical Linux concepts. Instead of just memorizing commands from a textbook, Bandit forced me to think critically, chain commands together, and solve puzzles in a real terminal environment. This hands-on approach builds the muscle memory and problem-solving mindset essential for a DevOps engineering path.

## Key Skills Gained

Throughout Levels 1 to 20, I developed and refined several core Linux skills:

- **File Navigation**: Deep understanding of absolute vs. relative paths, hidden files, and navigating complex directory structures.
- **Permissions & Ownership**: Grasping the Unix permissions model (`rwx`), understanding file ownership, and reading files under restricted permissions.
- **Searching**: Utilizing tools like `find` and `grep` to locate files based on specific properties (size, type, permissions) or search for patterns within text.
- **Data Processing**: Using commands like `sort`, `uniq`, and `strings` to filter, process, and extract meaningful data from noisy files or binaries.
- **Encoding/Decoding**: Recognizing and translating different data encodings like Base64, Hexadecimal, and ROT13 ciphers using command-line utilities.
- **SSH and Basic Networking**: Regularly establishing secure remote connections, managing keys, and understanding client-server interactions.

## Disclaimer

To respect the OverTheWire community guidelines and preserve the learning experience for others, **this documentation does not contain exact solutions, direct commands revealing the flag, or passwords.** Instead, it focuses on the underlying concepts, the commands utilized, and my personal takeaways from each challenge.
