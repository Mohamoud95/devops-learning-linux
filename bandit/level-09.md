# Level 8 to Level 9

## Goal
Find the only unique line of text in a file where every other line is duplicated.

## Concept Explanation
Data processing often involves sorting and finding unique entries. You can chain commands together using pipes (`|`) to pass the output of one command as the input to the next.

## Commands Used
- `sort`, `uniq`

## What I Learned
I learned the concept of command piping. Seeing how modular Linux commands can be chained to solve complex problems was a great 'aha' moment.
