# Level 15 to Level 16

## Goal
Submit the current password to a specific network port, but this time the connection must be encrypted using SSL/TLS.

## Concept Explanation
Standard network connections send data in plain text. Secure services require SSL/TLS encryption. You need specialized tools to establish these encrypted connections manually.

## Commands Used
- `openssl`

## What I Learned
I learned how to manually negotiate a secure SSL connection to interact with a secure service, an essential skill for debugging HTTPS and secure endpoints.
