# Level 1 to Level 2

## Goal
Read the contents of a file named with a special character (a dash `-`).

## Concept Explanation
Some file names can trick the command line into interpreting them as flags or arguments instead of file names. You must use specific pathing to tell the shell to treat it as a literal file.

## Commands Used
- `cat`, `./`

## What I Learned
This challenge taught me how important it is to understand absolute and relative paths when dealing with awkwardly named files.
