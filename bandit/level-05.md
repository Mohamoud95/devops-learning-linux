# Level 4 to Level 5

## Goal
Find the only human-readable file in a directory containing multiple files.

## Concept Explanation
Sometimes directories are filled with unreadable binary data. You can use commands that determine the type of data a file contains without opening it.

## Commands Used
- `file`, `cat`

## What I Learned
I learned how to determine file types programmatically, which is incredibly useful when dealing with unknown file structures.
