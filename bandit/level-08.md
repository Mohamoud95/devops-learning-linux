# Level 7 to Level 8

## Goal
Find a specific password stored next to the word 'millionth' in a very large text file.

## Concept Explanation
Opening a massive file manually is inefficient. Instead, command-line utilities allow you to search for specific text patterns within files instantly.

## Commands Used
- `grep`

## What I Learned
I experienced how crucial text filtering tools are for finding needles in haystacks. `grep` is a foundational tool for parsing logs.
