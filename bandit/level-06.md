# Level 5 to Level 6

## Goal
Find a specific file hidden somewhere under a directory based on its exact properties (size, non-executable, human-readable).

## Concept Explanation
When you have a deep directory structure and specific constraints, manual searching is impossible. You need powerful search utilities that can filter by file size and properties.

## Commands Used
- `find`, `cat`

## What I Learned
This was my first deep dive into the power of the `find` command, allowing me to search efficiently based on file metadata rather than just names.
