# Level 14 to Level 15

## Goal
Submit the current password to a specific network port on the local machine to retrieve the next password.

## Concept Explanation
Services run on specific network ports. You can interact with these services directly by opening raw network connections from the command line.

## Commands Used
- `nc` (netcat)

## What I Learned
I used netcat to interact with an open port, highlighting how basic network troubleshooting and service interaction works in Linux.
