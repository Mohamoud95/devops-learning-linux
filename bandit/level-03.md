# Level 2 to Level 3

## Goal
Read a file that has spaces in its filename.

## Concept Explanation
The command line interprets spaces as separators between commands and arguments. To handle filenames with spaces, you need to either use quotation marks or escape the spaces with a backslash.

## Commands Used
- `cat` with escaping or quotes

## What I Learned
I learned how shell interpretation works and the importance of escaping special characters to accurately target files.
