# Level 13 to Level 14

## Goal
Log into the next level using a private SSH key instead of a password.

## Concept Explanation
Passwords are not the only (or most secure) way to authenticate. Public/Private key pairs are the industry standard for secure, automated remote access.

## Commands Used
- `ssh` with key flag

## What I Learned
I learned how SSH keys function in practice, which is critical for secure server administration and automated CI/CD pipelines.
